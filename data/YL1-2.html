A Simulation Study: Health Biomarker

Yang Li

2026-05-22

Simulation

Suppose a public health agency is monitoring a continuous health biomarker, such as an inflammatory marker, air-pollution exposure biomarker, or disease-risk score, across a geographic region. Because it is expensive to collect samples everywhere, measurements are only taken at a limited number of clinics. The goal is to use these observed biomarker values to estimate the spatial pattern of health risk across the entire region.

In this simulation study, we generated data from \(n=80\) clinic locations randomly distributed over a \(100\times 100\) spatial region. Each clinic represents a sampling site where a health-related biomarker is measured.

The biomarker was assumed to have an overall average level of 10, with spatially correlated variation across the region. This means that clinics located near each other tend to have more similar biomarker values than clinics far apart.

Simulate Clinic Locations

We simulate 80 clinics whose positions are randomly distributed on a \(100\times 100\) region.

n <- 80  # number of clinics

sim_locs <- data.frame(
    clinic_id = 1:n,
    x = runif(n, 0, 100),
    y = runif(n, 0, 100)

Define a Spatial Covariance Model

We use grf function to simulate the data. An exponential covariance function is used.

sim <- grf(
  n = nrow(sim_locs),
  grid = sim_locs[, c('x', 'y')],
  cov.model = "exponential",
  cov.pars = c(4, 25),
  nugget = 0.5,
  mean = 10
)
## grf: simulation on a set of locations provided by the user
## grf: process with  1  covariance structure(s)
## grf: nugget effect is: tausq= 0.5 
## grf: covariance model 1 is: exponential(sigmasq=4, phi=25)
## grf: decomposition algorithm used is:  cholesky 
## grf: End of simulation procedure. Number of realizations: 1
sim_data <- data.frame(
  x = sim$coords[, 1],
  y = sim$coords[, 2],
  biomarker = sim$data
)

Plot and Fit the Empirical Variogram

First we estimate the empirical variogram \(\hat{\gamma}(h)=\frac{1}{N(h)}\sum_{N(h)}(Z(s_i)-Z(s_j))^2\). This is automated by the variogram() function.

coordinates(sim_data) <- ~ x + y  # convert to a spatial object

emp_vario <- variogram(biomarker ~ 1, data = sim_data)

plot(emp_vario, main = "Empirical Variogram")  # plot the empirical variogram

Next, we will fit the empirical variogram.

initial_model <- vgm(
    psill = 4,
    model = "Exp",
    range = 25,
    nugget = 0.5
)

fit_vario <- fit.variogram(emp_vario, initial_model)

fit_vario
##   model     psill    range
## 1   Nug 0.3320339  0.00000
## 2   Exp 3.1840686 17.33822
plot(emp_vario, fit_vario, main = "Fitted exponential variogram")

Create Prediction Grid and Do Ordinary Kriging

We make prediction at all lattice points of an \(100\times 100\) grid. In total, there are \(101\times 101\) predicted values.

grid <- expand.grid(
    x = seq(0, 100, length.out = 101),
    y = seq(0, 100, length.out = 101)
)

coordinates(grid) <- ~ x + y
gridded(grid) <- TRUE

We use ordinary kriging to get the predicted values.

ok_result <- krige(
    biomarker ~ 1,
    locations = sim_data,
    newdata = grid,
    model = fit_vario
)
## [using ordinary kriging]
ok_df <- as.data.frame(ok_result)

Plot the map of kriging variance

The kriging variance measures the uncertainty of the kriging prediction at each location. A small kriging variance means

  • prediction is more reliable, and
  • nearby observed data provide strong information.

A large kriging variance means

  • prediction is more uncertain, and
  • the location is far from observed data or poorly informed.
ggplot(ok_df, aes(x = x, y = y, fill = var1.var)) +
  geom_raster() +
  coord_equal() +
  labs(
    title = "Prediction variance map",
    x = "x",
    y = "y",
    fill = "Prediction\nvariance"
  ) +
  theme_minimal()
