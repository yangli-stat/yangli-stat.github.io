Introduction

We are considering the Jura soil geochemistry dataset in the gstat package. It includes concentrations of several heavy metals, such as cadmium, cobalt, chromium, copper, nickel, lead, and zinc.

In this analysis, we focus on cadmium concentration in topsoil.

The main goals are to:

  • explore the spatial distribution of cadmium concentration;
  • compute and model the empirical variogram;
  • perform ordinary kriging;
  • map the kriging prediction surface;
  • map the kriging prediction variance;
  • validate predictions using a separate validation dataset;
  • optionally compare with universal kriging using covariates.

coordinates(train_sp) <- ~ Xloc + Yloc
coordinates(valid_sp) <- ~ Xloc + Yloc
coordinates(grid_sp) <- ~ Xloc + Yloc

gridded(grid_sp) <- TRUE

Now:

  • train_sp contains the observed training data;
  • valid_sp contains validation locations;
  • grid_sp is the prediction grid.

Empirical Semivariogram

The empirical semivariogram is defined as

\[ \hat{\gamma}(h) = \frac{1}{2N(h)} \sum_{(i,j)\in N(h)} \left[ Z(s_i)-Z(s_j) \right]^2, \]

where \(N(h)\) is the set of pairs of locations separated by distance approximately \(h\).

emp_vario <- variogram(
  log_Cd ~ 1,
  data = train_sp
)

plot(
  emp_vario,
  main = "Empirical Variogram of log(Cd)"
)

The formula log_Cd ~ 1 means that we assume an unknown but constant mean over space. This corresponds to ordinary kriging.

Fit a Variogram Model

We fit an exponential variogram model.

initial_model <- vgm(
  psill = 0.4,
  model = "Exp",
  range = 1.0,
  nugget = 0.05
)

fit_vario <- fit.variogram(
  emp_vario,
  model = initial_model
)

fit_vario
##   model     psill     range
## 1   Nug 0.1940533 0.0000000
## 2   Exp 0.3524335 0.4649512

Plot the empirical variogram together with the fitted model.

plot(
  emp_vario,
  fit_vario,
  main = "Empirical Variogram with Fitted Exponential Model"
)

The exponential variogram model can be written as

\[ \gamma(h) = \tau^2 + \sigma^2 \left[ 1-\exp\left(-\frac{h}{\phi}\right) \right], \]

where:

  • \(\tau^2\) is the nugget effect;
  • \(\sigma^2\) is the partial sill;
  • \(\phi\) is the range parameter.

Ordinary Kriging

Ordinary kriging predicts the value at an unsampled location \(s_0\) using

\[ \hat{Z}(s_0) = \sum_{i=1}^{n} \lambda_i Z(s_i), \]

subject to the constraint

\[ \sum_{i=1}^{n} \lambda_i = 1. \]

The weights \(\lambda_i\) are determined using the fitted variogram model.

ok_result <- krige(
  formula = log_Cd ~ 1,
  locations = train_sp,
  newdata = grid_sp,
  model = fit_vario
)

The prediction variance is usually smaller near observed sampling locations and larger in areas farther from observations.

Back-Transform Predictions

Because the model was fitted on the log scale, the predictions are also on the log scale.

A simple back-transformation is

\[ \widehat{\text{Cd}}(s_0) = \exp \left[ \widehat{\log(\text{Cd})}(s_0) \right]. \]

ok_df$Cd_pred <- exp(ok_df$var1.pred)

ggplot() +
  geom_raster(
    data = ok_df,
    aes(x = Xloc, y = Yloc, fill = Cd_pred)
  ) +
  geom_point(
    data = train_df,
    aes(x = Xloc, y = Yloc),
    color = "black",
    size = 1.2
  ) +
  scale_fill_viridis(name = "Predicted\nCd") +
  coord_equal() +
  labs(
    title = "Back-transformed cadmium prediction surface",
    x = "X coordinate",
    y = "Y coordinate"
  ) +
  theme_minimal()

This simple transformation gives predictions on the original cadmium scale.

A technical note: the simple exponential back-transformation gives a median-type prediction on the original scale. A bias-corrected back-transformation may be used if the mean on the original scale is required.

Validation Using the Validation Dataset

The Jura dataset includes a separate validation dataset. We use it to evaluate prediction performance.

valid_pred <- krige(
  formula = log_Cd ~ 1,
  locations = train_sp,
  newdata = valid_sp,
  model = fit_vario
)
## [using ordinary kriging]
valid_pred_df <- as.data.frame(valid_pred)
valid_df <- as.data.frame(valid_sp)

validation_results <- data.frame(
  observed = valid_df$log_Cd,
  predicted = valid_pred_df$var1.pred
)

validation_results$error <- validation_results$observed -
  validation_results$predicted

mean_error <- mean(validation_results$error)

Universal Kriging with Covariates

The Jura dataset includes covariates such as land use and rock type. These can be used in universal kriging.

Universal kriging allows the mean structure to depend on covariates:

\[ Z(s) = \beta_0 + \beta_1 X_1(s) + \beta_2 X_2(s) + \varepsilon(s). \]

Here, we include Landuse and Rock as covariates.

emp_vario_uk <- variogram(
  log_Cd ~ Landuse + Rock,
  data = train_sp
)

initial_model_uk <- vgm(
  psill = 0.3,
  model = "Exp",
  range = 1.0,
  nugget = 0.05
)

fit_vario_uk <- fit.variogram(
  emp_vario_uk,
  model = initial_model_uk
)

plot(
  emp_vario_uk,
  fit_vario_uk,
  main = "Variogram for Universal Kriging Model"
)

Perform universal kriging.

uk_result <- krige(
  formula = log_Cd ~ Landuse + Rock,
  locations = train_sp,
  newdata = grid_sp,
  model = fit_vario_uk
)
## [using universal kriging]
uk_df <- as.data.frame(uk_result)

Plot the universal kriging prediction surface.

ggplot() +

Compare Ordinary and Universal Kriging

We can compare the two models using the validation dataset.

uk_valid_pred <- krige(
  formula = log_Cd ~ Landuse + Rock,
  locations = train_sp,
  newdata = valid_sp,
  model = fit_vario_uk
)
## [using universal kriging]
uk_valid_pred_df <- as.data.frame(uk_valid_pred)

uk_validation_results <- data.frame(
  observed = valid_df$log_Cd,
  predicted = uk_valid_pred_df$var1.pred
)

uk_validation_results$error <- uk_validation_results$observed -
  uk_validation_results$predicted

comparison <- data.frame(
  Model = c("Ordinary kriging", "Universal kriging"),
  Mean_Error = c(
    mean(validation_results$error),
    mean(uk_validation_results$error)
  ),
  RMSE = c(
    sqrt(mean(validation_results$error^2)),
    sqrt(mean(uk_validation_results$error^2))
  ),
  MAE = c(
    mean(abs(validation_results$error)),
    mean(abs(uk_validation_results$error))
  )
)

comparison
##               Model  Mean_Error      RMSE       MAE
## 1  Ordinary kriging -0.05216502 0.5651087 0.4500128
## 2 Universal kriging -0.05583267 0.5736756 0.4550778

The model with smaller RMSE and MAE has better predictive performance on the validation dataset. The two kriging methods have similar performance.
