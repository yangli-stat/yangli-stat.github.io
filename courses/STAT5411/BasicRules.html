Basic rules

SAS University Edition can be installed free of charge at https://www.sas.com/en_us/software/university-edition.html.

  • SAS variable names must be 32 characters or less, consisting of letters (A to Z, a to z), digits (0 to 9), and the underscore character (_). The first character must be a letter or an underscore. Spaces cannot appear in SAS names.
    • Valid names: Stat5411, _ANOVA, R1d934jd8
    • Invalid names: 5411STAT, Math%101, Math and Stat
  • SAS is NOT case sensitive. Therefore, ANOVA, anova, aNovA are exactly the same in SAS.
  • Each statement in SAS must end in a semicolon (;).
  • Lines beginning with an asterisk (*) are treated as comments. You can also enclose comments between /* and */.
    • * This whole line is a comment.
    • /* This is another comment */
  • Data steps begin with data and procedure steps begin with proc.
  • Check log file for more information (warnings, errors, ...).
