Now suppose there are \(t>2\) populations (treatments). Each treatment has \(n\) observations so that the total number of observations is \(nt\). First we want to see if these \(t\) treatments have the same mean values \(\mu_1=\mu_2=\cdots = \mu_t\). Let's use the data set in Example 8.4 of the textbook.

To input the data set into SAS, there needs to be a categorical variable representing the treatments.

    data example84;
        input y trt $;
        datalines;
        1 1km
        5 1km
        2 1km
        1 1km
        2 1km
        2 1km
        4 1km
        3 1km
        0 1km
        2 1km
        4 5km
        8 5km
        2 5km
        3 5km
        8 5km
        5 5km
        6 5km
        4 5km
        3 5km
        3 5km
        20 10km
        26 10km
        24 10km
        11 10km
        28 10km
        20 10km
        19 10km
        19 10km
        21 10km
        24 10km
        37 20km
        30 20km
        26 20km
        24 20km
        41 20km
        25 20km
        36 20km
        31 20km
        31 20km
        33 20km
        ;
    run;

Here, since trt is a variable with characters (1km, 5km, 10km, 20km), we need to add $ symbol when we read the data. $ tells SAS to start reading a character variable for trt.

The ANOVA table can be generated using proc glm procedure. glm represents generalized linear models. Here class indicates that trt is a classification variable (four levels of the factor). model specifies the ANOVA model \(y_{ij}=\mu+\tau_i + \epsilon_{ij}\). All terms in the model need to be explicitly specified except for the overall mean \(\mu\) and the random error \(\epsilon\) which will be added automatically by SAS. Since \(\tau_i\) is the treatment effect, we need to use model y = trt for the model specification.

    proc glm data=example84;
        class trt;
        model y = trt;
    run;
