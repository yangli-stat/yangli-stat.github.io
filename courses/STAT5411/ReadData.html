Read data into SAS

Previous   Back   Next

There are multiple ways of reading a data set into SAS for the subsequent data analysis. The following three methods are the most commonly used.

Read from a local file

You can read the data from a file on the local drive. You have to make sure mydata.txt is on the local drive.

data set1;
  infile 'mydata.txt';
  input v1 v2 v3;
run;

Read from an online file

If the file is saved online under an url, you can use filename to access it.

filename file1 url 'http://www.somewebsite.com/mydata.txt';
data set1;
  infile file1;
  input v1 v2 v3;
run;

Input data directly in SAS

If the data set is small, it is in general easier to input the data set directly in SAS using datalines. 

data set1;
  input v1 v2 v3;
  datalines;
  3 1 2
  4 2 7
  ;
run;

Here, set1 is the the assigned name of the data set. It contains three variables, v1, v2, and v3. By default, SAS starts reading data from the first row. You can use infile file1 firstobs=2 to skip the first row which might contain the variable names.
