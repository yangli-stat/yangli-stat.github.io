Generate summary statistics

To analyze a data set, it is important to first have some summary statistics (mean, variance, etc) for the variables you are interested in. In doing this, you will have some basic ideas about the variables. It is also effective to find errors in the data set.

We will use cars.txt for illustration. This file contains four variables on 32 different car models.

  • mpg: mileage per gallon
  • cyl: number of cylinders
  • hp: horse power
  • wt: weight

First, we need to read the file into SAS. The file is already saved at http://www.d.umn.edu/~yangli/SAS/cars.txt. You can use it directly.

    filename file1 url 'http://www.d.umn.edu/~yangli/5411/data/cars.txt';
    data cars;
        infile file1 firstobs=2;
        input mpg cyl hp wt;
    run;

After reading the data, it is always a good practice to check if everything is fine by using print procedure to show the whole data set.

    proc print data=cars;
    run;

You will get the following table which looks great (32 rows, 4 columns, no missing values).

Graphs

Now we are ready to make some simple but illustrative graphs. They can be produced easily without writing any code. On the left panel, go to "Tasks and Utilities" -> "Tasks" -> "Graph", and you will see a list of graphing tools as shown below.

After your selection and input of all required information, SAS will automatically generate code for you. For example, if you choose "Scatter Plot", and WORK.CARS in "DATA", mpg as "X axis", wt as "Y axis", you will get

    proc sgplot data=WORK.CARS;
        scatter x=mpg y=wt /;
        xaxis grid;
        yaxis grid;
    run;

Summary statistics

Summary statistics are used to summarize the properties of a group of observations, in order to convey a large amount of information with only a few values. Commonly used summary statistics include

  • measures of location: mean, median, mode
  • measures of dispersion: variance, standard deviation, range, interquartile range
  • measures of shape of the distribution: skewness, kurtosis
  • measures of relationship between more than one variables: covariance, correlation coefficient

Many of these measures can be obtained with one simple SAS command. Suppose we are interested in wt variable in the dataset. Using

    proc univariate data=cars;
        var wt;
    run;

we can get the following output.

You can also use means procedure to get the same results. Here, options n mean max min range std tell SAS to only generate these summary statistics to avoid very bulky output.

    proc means data=cars n mean max min range std;
        var wt;
    run;
