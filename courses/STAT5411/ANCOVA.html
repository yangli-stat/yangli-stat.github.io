We are going to use Example 16.1 to illustrate how analysis of covariance (ANCOVA) works.

There are three models to fit in the study.

  1. Full model (with everything):
    2. $$ y_{ij}=\beta_0 + \tau_i + \beta_1 x_{ij} + \epsilon_{ij} $$
  2. Reduced model I (without covariate):
    3. $$ y_{ij}=\beta_0 + \tau_i + \epsilon_{ij} $$
  3. Reduced model II (without treatment differences):
    4. $$ y_{ij}=\beta_0 + \beta_1 x_{ij} + \epsilon_{ij} $$

where \(\tau_i\) is the treatment effect (two types of fertilizers and one control), \(x_{ij}\) is the covariate (height).

    filename file1 url 'http://www.d.umn.edu/~yangli/5411/data/fertilizer.txt';
    data fertilizer;
      infile file1 firstobs=2;
      input yield height trt $;
    run;
    
    /* Full Model */
    proc glm data=fertilizer;
        class trt;
        model yield = height trt / solution;
    run;
    
    /* Reduced Model I */
    proc glm data=fertilizer;
        class trt;
        model yield = trt / solution;
    run;
    
    /* Reduced Model II */
    proc glm data=fertilizer;
        class trt;
        model yield = height / solution;
    run;
