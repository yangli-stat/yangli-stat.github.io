Randomized complete block design with one factor

Let's use the data set of Example 15.1 from the textbook.

We will still use glm procedure to analyze the data.

    filename myfile url 'http://www.d.umn.edu/~yangli/5411/data/insecticides.txt';

    data insecticides;
        infile myfile firstobs = 2;
        input yield insecticide block;
    run;

    proc glm data = insecticides;
        class block insecticide;
        model yield = insecticide block;
    run;

Here the model is written as yield = insecticide block because there is no interaction between insecticide and block. The output is

The test for three insecticides \(H_0: \tau_1=\tau_2=\tau_3\) gives a \(p\)-value of 0.0001. The conclusion is that there is strong difference among three types of insecticides.

Latin square design

Let's use Example 15.5 from the textbook which is also covered in the lecture.

The model is \(y_{ijk}=\mu+\tau_k+\beta_i+\gamma_j+\epsilon_{ijk}\) where \(\tau_k\) is the treatment effect; \(\beta_i\) is the row effect; \(\gamma_j\) is the column effect.

    data latinsquare;
        input y month $ house $ trt $;
        datalines;

    328 m5 h1 b5
    205 m5 h2 b1
    321 m5 h3 b2
    208 m5 h4 b3
    333 m5 h5 b4
    ;
    run;
    
    proc glm data=latinsquare plots(unpack)=diagnostics;
        class month house trt;
        model y = month house trt;
    run;

Here using plots(unpack)=diagnostics will generate several diagnostics plots including residual plot, qq plot, etc.
