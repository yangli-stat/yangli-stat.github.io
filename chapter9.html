  • Decision trees are a non-parametric supervised learning method used for classification and regression.

Decision Tree

  • Some notations

Decision Tree

  • The algorithm of decision tree models works by repeatedly partitioning the data into multiple sub-spaces, so that the outcomes in each final sub-space is as homogeneous as possible.

  • This procedure is called recursive partitioning.

  • Tree model works for both regression and classification problems.

    • Regression tree: continuous response variable

    • Classification tree: categorial response variable

  • The decision rules generated by model are generally visualized as a binary tree.

Regression Tree with tree

  • Lets use the Boston data set from earlier chapters. First, lets use the tree function in the tree package.

    library(ISLR2)
library(tree)
set.seed(123)
train <- sample(1:nrow(Boston), round(nrow(Boston)*0.8))
train.boston <- Boston[train, ]
test.boston <- Boston[-train, ]
tree.boston <- tree(medv ~ ., Boston, subset=train)
# this is the same as
# tree(medv ~ ., train.boston)

Classification Tree with rpart

  • We can use rpart as a classifier.

    iris.model <- rpart(Species ~., data=iris)
par(xpd=NA) # otherwise on some devices the text is clipped
plot(iris.model)
text(iris.model)

  • The rules in the tree can be printed out.

    print(iris.model, digits=2)
